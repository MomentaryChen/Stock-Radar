# Project Rules

## Language Policy

- All source code must be written in English
- All code comments must be in English
- All variable names, function names, class names, and identifiers must be in English
- All commit messages must be in English
- All docstrings and inline documentation must be in English
- Log messages and error messages must be in English
- Only user-facing UI text (labels, tooltips) may use other languages when required by the product
